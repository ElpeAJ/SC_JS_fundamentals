# SC_JS_fundamentals

# Lesson 1: JavaScript Fundamentals and User Input 🚀

Welcome to Day 1 of JavaScript! In this lesson, we cover the core syntax for writing JavaScript code, declaring variables, taking user input, and joining text together.

## ⚠️ Getting Started (Important Instructions)
To work on these tasks, you must set up your own copy of the project:
1. Click the **"Fork"** button at the top right of this repository to create a copy under your own GitHub account.
2. **Clone** your personal forked repository to your local machine using your terminal:
   git clone <your-forked-repo-url>
3. Create your files and practice the tasks locally on your computer.
4. **DO NOT create a Pull Request (PR)** back to the main classroom repository. All your work stays in your own personal repository!

## 📝 What You Will Learn
* How to add JavaScript directly inside an HTML file
* How to link an external JavaScript file
* How to use the pop-up alert and prompt functions
* How to create variables to store data
* The difference between string concatenation and template literal interpolation
* How to write code comments using keyboard shortcuts

## 🛠️ Classroom Tasks

1. **Internal JavaScript:** Learn how to write code inside <script> tags directly in your HTML file.
2. **External JavaScript:** Link a separate script.js file to your HTML file using the script source attribute.
3. **The Alert Function:** Use the alert() function to display a "Hello World" pop-up message in the browser.
4. **Variables Setup:** Practice declaring variables with let and initializing them with data.
5. **User Input:** Take real-time input from a user using the prompt() function.
6. **Data Storage:** Save the user's prompt input safely inside a custom variable.
7. **Displaying Input:** Fire an alert popup that displays the exact message or name the user typed in.
8. **Text Manipulation:** Practice combining text strings using two methods:
   * Concatenation: Joining strings using the plus (+) sign.
   * Interpolation: Injecting variables directly into text using backticks (``) and ${variable} syntax.
9. **Commenting Code:** Master the shortcut (Ctrl + /) to instantly comment out lines of code.

## 🚀 How to Run the Code
1. Open your project folder in VS Code.
2. Open index.html and write your HTML layout.
3. Open script.js and write your JavaScript automation.
4. Double-click your index.html file to open it in any web browser and watch your script run!
